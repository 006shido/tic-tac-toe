# Tic-Tac-Toe in C

A simple command-line Tic-Tac-Toe game written in C. This project allows two players to play against each other in the terminal using a 3×3 game board.

## Features

- Two-player gameplay
- Interactive board display
- Input validation for occupied or invalid positions
- Automatic win detection
- Draw detection
- Lightweight and beginner-friendly code

## How to Play

1. Run the program.
2. Player 1 uses **X**.
3. Player 2 uses **O**.
4. Enter a position number (1-9) to place your mark.
